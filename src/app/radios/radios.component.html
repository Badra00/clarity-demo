Radio groups

Suppose we have some sort of data structure like:

  
    {{dataCode}}

and we want to let users select one of the turtles with a radio group. The number of items in the data might change, so we need to 1) dynamically generate a radio group and 2) bind to the current selection.

This took me a while to figure out, but a working example which binds to the selected object is below. It's also possible to bind only to the Name property or similar. The only minor annoyance is that providing an initial value doesn't seem to work when binding to the object.
